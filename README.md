# 🔐 LocalPasses

**Локальный менеджер паролей без облака, подписок и слежки.**


## 📌 О проекте

**LocalPasses** — это простой и безопасный менеджер паролей, который хранит все данные **только на вашем компьютере**. 

Никакого облака. Никаких серверов. Никаких подписок.

Все пароли шифруются мастер-паролем (AES-256-GCM) и расшифровываются только при вводе мастер-пароля. Без него данные невозможно прочитать — даже самому приложению.


## ✨ Особенности

- 🔒 **Шифрование AES-256-GCM** — надёжное шифрование всех данных
- 🔑 **Один мастер-пароль** — ключ ко всем паролям
- 💻 **Локальное хранение** — данные не покидают ваш компьютер
- 🚀 **Не требует интернета** — работает офлайн
- 💰 **Полностью бесплатно** — никаких подписок
- 🎨 **Тёмная тема** — приятный интерфейс
- 🔐 **Генератор паролей** — создание сложных паролей
- 📂 **Хранение в AppData** — данные в скрытой папке пользователя


## 📥 Установка

1. Скачайте `LocalPasses_Setup.exe` из [Releases](https://github.com/rabbitboxy/LocalPasses/releases)
2. Запустите установщик
3. Следуйте инструкциям
4. Готово!


## 💾 Бекап и хранение данных

**Все данные хранятся только у вас на компьютере в папке:**

C:\Users\ВАШЕ_ИМЯ\AppData\Roaming\LocalPasses

**Как открыть папку:**
1. Нажмите `Win + R`
2. Введите `%APPDATA%\LocalPasses`
3. Нажмите Enter


## ⚠️ ВАЖНО!

**Забыли мастер-пароль** Пароли потеряны **НАВСЕГДА** ❌ 
**Не сделали бекап** При сбое диска пароли потеряны ❌
**Сделали бекап + помните пароль** Пароли восстановлены ✅

**Обязательно делайте бекап!**

Просто скопируйте папку `%APPDATA%\LocalPasses` на флешку или в облако.
